trial-scala-parser
==================

Scalaのパーサコンビネータについて勉強する。

操作
----

プロジェクトの実行

```sh
$ sbt run
```


参考
----

### 準備

初期プロジェクトの作成に使用したコマンドライン

```sh
$ sbt new scala/scala-seed.g8
```
